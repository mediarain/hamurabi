Hamurabi Alexa
===============
A Node application allowing Amazon Alexa users to play the classic [Hamurabi](http://www.hammurabigame.com/hammurabi-game.php) terminal game.

Development Setup
-------------
 * `npm install`
 * `npm test`
 * Install `ask-cli` via `npm install ask-cli`, then run `ask init`
 * `ask deploy`
